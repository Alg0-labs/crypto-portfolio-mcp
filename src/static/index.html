Get started

Connect your AI assistant

Works with any MCP client. Pick yours, paste the config, done.

1

Grab a free Moralis API key at moralis.io — needed only for the 4 wallet tools (market data needs no key).

2

Add the config below to your client, replacing YOUR_MORALIS_KEY with your key.

3

Restart the client and ask: "What's the price of ETH?" or "Analyze wallet 0x… on Ethereum."

{
  "mcpServers": {
    "crypto-portfolio": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "__MCP_URL__",
        "--header",
        "X-Moralis-Key:YOUR_MORALIS_KEY"
      ]
    }
  }
}

Add to claude_desktop_config.json → Settings ▸ Developer ▸ Edit Config. The mcp-remote bridge ships with npx; no install needed.

{
  "mcpServers": {
    "crypto-portfolio": {
      "url": "__MCP_URL__",
      "headers": {
        "X-Moralis-Key": "YOUR_MORALIS_KEY"
      }
    }
  }
}

For clients that support remote MCP servers with custom headers directly.

MCP endpoint   __MCP_URL__
Transport      Streamable HTTP (stateless)
Auth header    X-Moralis-Key: YOUR_MORALIS_KEY   (portfolio tools only)
Health check   __ORIGIN__/health

We store no keys. Each request carries the caller's own key, so usage is always on your account.
